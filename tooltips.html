Tooltips

Tooltips identify elements or provide additional, useful information about the referring elements. Tooltips are different from ALT-attributes, which are intended primarily for static images. A tooltip fades in upon hover or focus on the trigger element. It remains open until the cursor moves outside of itself or the trigger, or focus is moved away from the trigger.

On this page
  • Default filled buttons
  • Large outline buttons
  • Default filled buttons with icons

/* Tooltip */

div.shr_tooltip_text {
    position: relative;
}

div.shr_tooltip_text span {
    display: none;
    position: absolute;
    top: -5px;
    left: 60px;
    width: 15.625rem;
    padding: 0.625rem;
    z-index: 100;
    background: var(--shr_primary_black_default);
    color: #fff;
    -moz-border-radius: 0.625rem;
    /* this works only in camino/firefox */
    -webkit-border-radius: 0.625rem;
    /* this is just for Safari */
}

div.shr_tooltip_text span:before {
    content: '';
    display: block;
    width: 0;
    height: 0;
    position: absolute;
    border-top: 8px solid transparent;
    border-bottom: 8px solid transparent;
    border-right: 8px solid var(--shr_primary_black_default);
    left: -8px;
    top: 7px;
}

div:hover.shr_tooltip_text {
    font-size: 100%;
    /* this is just for IE */
}

div:hover.shr_tooltip_text span {
    display: block;
}
