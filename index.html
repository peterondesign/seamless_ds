Buttons

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

On this page
  • Default filled buttons
  • Large outline buttons
  • Default filled buttons with icons
  • Button CSS

Examples

Default filled buttons

<button type="button" class="shr_button_defaultsize shr_button_success_filled">Success</button>
<button type="button" class="shr_button_defaultsize shr_button_alarm_filled"><img src="icon.svg" class="shr_button_icon"></img>Alarm</button>

Button CSS

                
/* Buttons */

.shr_button_defaultsize,
.shr_button_largesize {
  display: inline-block;
  font-weight: 400;
  line-height: 1.5;
  text-align: center;
  text-decoration: none;
  vertical-align: middle;
  cursor: pointer;
  -webkit-user-select: none;
  -moz-user-select: none;
  user-select: none;
  background-color: transparent;
  border: 1px solid transparent;
  font-size: 1rem;
  border-radius: 0.25rem;
  transition: color 0.15s ease-in-out, background-color 0.15s ease-in-out,
    border-color 0.15s ease-in-out, box-shadow 0.15s ease-in-out;
  margin-right: 15px;
}
@media (max-width: 990px) {
  .shr_button_defaultsize,
  .shr_button_largesize {
    margin-bottom: 15px;
    width: 100%;
  }
}
@media (prefers-reduced-motion: reduce) {
  .shr_button_defaultsize,
  .shr_button_largesize {
    transition: none;
  }
}
