Tabs

Tabs are used to divide content into meaningful, related sections. Tabs allow users to focus on one specific view at a time while maintaining sight of all the relevant content options available. Each tab, when active, will reveal its own unique content.

Tab CSS

                  
/* Tabs */

.shr_tabs_horizontal {
    display: flex;
    flex-direction: row;
}

.shr_tabs_horizontal>div {
    color: var(--shr_primary_black_lt1);
    border-bottom: 1px solid var(--shr_primary_blue_lt2);
    padding: 20px;
    cursor: pointer;
}

.shr_tabs_horizontal>.shr_tab_active {
    font-weight: 500;
    color: var(--shr_primary_blue_default);
    border-bottom: 4px solid var(--shr_primary_blue_default);
}

.shr_tabs_vertical {
    display: flex;
    flex-direction: column;
}

.shr_tabs_vertical>div {
    color: var(--shr_primary_black_lt1);
    border-right: 1px solid var(--shr_primary_blue_lt2);
    padding: 20px;
    cursor: pointer;
}

.shr_tabs_vertical>.shr_tab_active {
    font-weight: 500;
    color: var(--shr_primary_blue_default);
    border-right: 4px solid var(--shr_primary_blue_default);
}

.shr_tab_content {
    animation: shr_fadeEffect 1s;
    /* Fading effect takes 1 second */
}


/* Go from zero to full opacity */

@keyframes shr_fadeEffect {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}
